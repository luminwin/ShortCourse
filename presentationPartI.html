Brief Overview

randomForestSRC is a fast OpenMP and memory efficient package for fitting random forests (RF) for univariate, multivariate, unsupervised, survival, competing risks, class imbalanced classification and quantile regression. A basic grow call is of the form: rfsrc(formula, data, ntree, mtry, nodesize)

Brief Overview

rfsrc(formula, data, ntree, mtry, nodesize)

Grow your RF through rfsrc
Specify your model in formula
Provide your data frame in data
Tune your model via ntree, mtry, nodesize

Quick Start

Tip

The last line(s) will show the cross validated prediction performance

> 14                      (OOB) R squared: 0.7745644
> 15                     (OOB) Error rate: 245.3191
  • Model performance is displayed in the last two lines of the output in terms of out-of-bag (OOB) prediction error. In the above regression model, this is evaluated as the cross-validated mean squared error (MSE) estimated via the out-of-bag data shown in line 15
  • Since MSE is lack of scale invariance and interpretation, standardized MSE, defined as the MSE divided by the variance of the outcome, is used and converted to R squared or the percent of variance explained by a random forest model which has an intuitive and universal interpretation, shown in line 14

What is OOB prediction error?

  • Since each tree is grown from a random subset of the data, the package returns both out-of-sample and in-sample predicted values from the forest, where the former are calculated using the hold out data for each tree ($predicted.oob), and the latter are from the data used to train the tree ($predicted)
  • The out-of-sample values $predicted.oob should be used for inference on the training data since they are cross-validated and will not over-fit the data. See the Forest Weights, In-Bag (IB) and Out-of-Bag (OOB) Ensembles vignette for a more formal description of IB and OOB and how they are used to define the ensemble 
> 14                      (OOB) R squared: 0.7745644
> 15                     (OOB) Error rate: 245.3191

we can get the error rate (mean-squared error) directly from the OOB ensemble by comparing the response to the OOB predictor below

print(mean((airq.obj$yvar - airq.obj$predicted.oob)^2))
> [1] 245.3191

Learn more: https://www.randomforestsrc.org/articles/forestWgt.html

Data

Data structures allowed

We only allow data.frame format and you have to use the as.data.frame() commend to convert matrix, tibble, etc

Clean up your data

Choose your variables in formula and grow a tree

o <- rfsrc(y ~ a + z, data= dta, ntree = 1)

your outcome(s) will be saved in o$y and your predictors are in o$x from dta without missing values

Impute your data

To impute your data, use the following equivalent commend

o <- impute(y ~ a + z, data = dta)   
o <- rfsrc(y ~ a + z, data = dta, 
              na.action = "na.impute")
